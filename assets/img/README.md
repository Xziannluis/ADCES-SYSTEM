# Image assets

This folder contains static images used by the web UI and printable reports.

Recommended report header images:

- `smcc_seal.png` (SMCC seal / logo)
- `socotec_iso9001.png` (SOCOTEC ISO 9001 mark)
- `pab_ab.png` (PAB accredited mark)

If you add/replace images, keep filenames stable so templates don't need frequent edits.
